Service

MskSnackBarService is a singleton service for creating snackbar. Internally it uses MatSnackBar to create and show the snackbar.

Snackbar Config

Here is the interface for the snack bar configuration:

Methods

open(config: MskSnackBarConfig): MatSnackBarRef<MskSnackbarComponent>
Opens the snackbar with the given configuration
