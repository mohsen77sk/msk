Component

msk-alert is a customizable component for displaying all kinds of alerts across your app. They can be controlled from your components as well as from the MskAlertService.

Exported as: mskAlert

Properties

Name Description Default
name: <string> A unique name for the alert box to access it from MskAlertService. Auto generated
appearance: <MskAlertAppearance> Appearance of the alert box. soft
dismissed: <boolean> Whether the alert box is dismissed. false
dismissible: <boolean> Whether the alert box is dismissible. This must be true for dismissed to work. false
showIcon: <boolean> Whether the icon is shown. true
type: <MskAlertType> The type of the alert box. basic
afterDismissed: <true> An event emitted after the alert box dismissed. -
afterShown: <true> An event emitted after the alert box shown. -

Dismissible alerts

Dismissible alerts can be removed from the view by clicking their dismiss (x) button. In order to get the dismiss button visible, the [dismissed] input must be explicitly set either to true or false. Setting the [dismissed] input to null or simply removing it from the element will remove the dismiss button.
